# Weather App - The Odin Project

This is a weather application built as part of the [The Odin Project](https://www.theodinproject.com/) curriculum. The goal of this project was to practice using asynchronous JavaScript, specifically the Fetch API, to retrieve and display data from a third-party API.

## Features

- **Real-time Weather Data:** Fetches current weather conditions for any city worldwide.
- **Async/Await:** Utilizes modern JavaScript asynchronous patterns for clean and readable code.
- **Dynamic UI:** Updates the DOM dynamically based on API responses.
- **Search Functionality:** Allows users to search for specific locations.
- **Error Handling:** Basic validation and error catching for invalid city names or network issues.

## Built With

- **HTML5 & CSS3** - Semantic structure and custom styling.
- **JavaScript (ES6+)** - Logic, Fetch API, and DOM manipulation.
- **Webpack** - Module bundling and asset management.
- **ESLint & Prettier** - Code linting and formatting.
- **Weather API** - Data sourced from [WeatherAPI.com](https://www.weatherapi.com/) (or your chosen provider).
